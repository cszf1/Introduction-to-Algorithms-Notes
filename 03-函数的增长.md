## 本章导读：为什么学这一章？

### 🏃 生活类比：跑步速度比较

想象你和朋友在讨论跑步速度：

- 你说："我跑步速度大约是每秒8米"
- 朋友说："我比你快一点，大概每秒10米左右"

但在算法分析中，我们关心的不是具体数值，而是**增长趋势**。就像：

- 有人说："我是业余跑者" vs "我是专业运动员"
- 重点不在具体速度，而在**级别**

**渐近记号就是算法的"级别标签"**。它帮我们回答：
- 这个算法是 O(n) 级别还是 O(n²) 级别？
- 当数据量非常大时，哪个算法更快？

### 📊 为什么需要渐近分析？

回顾第2章，插入排序的运行时间可以写成：
```
T(n) = 2n² + 3n + 5
```

但我们通常只关心**最高阶项**，说它"是 Θ(n²)"，为什么？

**因为当 n 很大时，低阶项和常数项几乎不影响比较！**

```
n=1000时：
2n² = 2,000,000
3n   =     3,000
5    =         5
```

高阶项完全主导了结果。这就像：
- 比较大城市人口：北京 2000万 vs 上海 2500万
- 你不会把"外来人口 50万"算进去比较——太小了

## 3.1 渐近记号

### 🃏 生活类比：考试成绩评估

假设你考了若干次试，成绩波动很大：

| 次数 | 成绩 |
|:---:|:---:|
| 1 | 85分 |
| 2 | 72分 |
| 3 | 91分 |
| 4 | 78分 |
| 5 | 88分 |

**如何描述你的水平？**

- **Θ(85)**：你的成绩基本在85分左右晃，波动不大
- **O(90)**：你最多考到90分（上限）
- **Ω(70)**：你最少也有70分（下限）

这就是三种记号的直观含义！

---

### Θ记号（渐近紧确界）

#### 📝 定义

对于给定的函数 g(n)，用 **Θ(g(n))** 表示以下函数集合：

```
Θ(g(n)) = { f(n) : 
    存在正常量 c1、c2 和 n0，
    使得对所有 n ≥ n0，有：
    0 ≤ c1·g(n) ≤ f(n) ≤ c2·g(n)
}
```

#### 🎨 图解理解

```
f(n)
│                    ╱──────────── c2·g(n)
│                  ╱
│                ╱
│              ╱
│            ╱
│          ╱
│        ╱
│      ╱
│    ╱
│  ╱───────────────────── c1·g(n)
│╱
└─────────────────────────────→ n
         n0
```

**关键点：f(n) 被"夹在"两条曲线之间，像三明治！**

#### 🏠 生活类比：房租三明治

你付的房租 f(n)：
- 至少是基本生活费的 c1·g(n) 倍
- 最多是预算上限的 c2·g(n) 倍
- 当 n（居住时间）足够长后，基本稳定在这个范围内

#### 📐 形式化证明示例

**证明：12n² - 3n = Θ(n²)**

我们需要找到 c1、c2、n0，使得：
```
c1·n² ≤ 12n² - 3n ≤ c2·n²
```

两边除以 n²（n ≥ 1）：
```
c1 ≤ 12 - 3/n ≤ c2
```

取 c2 = 12（任何 ≥ 12 的数都行），右边对所有 n ≥ 1 成立

取 c1 = 1/14，则当 n ≥ 7 时，12 - 3/n ≥ 1/14 成立

所以取 **c1 = 1/14，c2 = 1/2，n0 = 7**，命题得证！

> 💡 **直觉验证**：去掉低阶项 -3n，忽略常数 12，得到 n²，正确！

#### ❌ 错误证明示例

**为什么 6n³ ≠ Θ(n²)？**

假设 6n³ = Θ(n²)，则存在 c、n0，使得：
```
6n³ ≤ c·n²  (对于所有 n ≥ n0)
```

两边除以 n²：
```
6n ≤ c
```

当 n > c/6 时，不等式不成立。**没有这样的常数 c 能约束所有 n！**

### O记号（渐近上界）

#### 📝 定义

```
O(g(n)) = { f(n) : 
    存在正常量 c 和 n0，
    使得对所有 n ≥ n0，有：
    0 ≤ f(n) ≤ c·g(n)
}
```

#### 🎨 图解理解

```
f(n)
│                             
│                           ╱── f(n) 的增长"触碰"天花板
│                         ╱    
│                       ╱      
│                     ╱        
│                   ╱          
│                 ╱            
│               ╱              
│             ╱                
│           ╱                  
│         ╱──────────────────── c·g(n) ← 上界
└─────────────────────────────────────→ n
         n0
```

**关键点：f(n) 永远不会超过 c·g(n) 这条线！**

#### 🏠 生活类比：堵车限速

- 高速公路限速 120km/h
- 你的车速 f(n) 永远不会超过 1.2×限速（常量倍）
- 所以你的速度 = O(120km/h)

#### 🔑 重要性质

1. **Θ(g(n)) ⊂ O(g(n))**：Θ 蕴含 O（紧确界当然是上界）
2. O 记号更常用，因为它只要求"不超过"，不要求"不低于"

**为什么用 O 而不是 Θ？**

有时候我们很难找到紧确下界，但很容易找到上界：
- "这个程序运行时间不超过 O(n²)"
- 实际可能是 Θ(n) 或 Θ(n log n)，但"不超过 n²"已经够用

#### 📐 形式化证明示例

**证明：n² = O(n²)**

取 c = 1，n0 = 1，则对所有 n ≥ 1：
```
n² ≤ 1·n²  ✓
```

**证明：n² = O(n³)**

取 c = 1，n0 = 1，则对所有 n ≥ 1：
```
n² ≤ 1·n³  ✓
```

> ⚠️ 虽然两个都正确，但 n² = O(n³) 是**非紧确上界**，不如 n² = Θ(n²) 有用

---

### Ω记号（渐近下界）

#### 📝 定义

```
Ω(g(n)) = { f(n) : 
    存在正常量 c 和 n0，
    使得对所有 n ≥ n0，有：
    0 ≤ c·g(n) ≤ f(n)
}
```

#### 🎨 图解理解

```
f(n)
│                                   
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ c·g(n) ← 下界
│           ╲              
│             ╲            
│               ╲          
│                 ╲        
│                   ╲      
│                     ╲    
│                       ╲  
│                         ╲──────────── f(n)
└─────────────────────────────────────→ n
         n0
```

**关键点：f(n) 永远不会掉到 c·g(n) 这条线以下！**

#### 🏠 生活类比：保底工资

- 公司承诺：每月至少发 c·g(n) = 3000元
- 你的实际工资 f(n) 永远 ≥ 3000元
- 所以你的工资 = Ω(3000)

#### 🔑 重要性质

1. **Θ(g(n)) ⊂ Ω(g(n))**：Θ 蕴含 Ω（紧确界当然也是下界）
2. Ω 记号描述算法复杂度的"最好情况"

#### 📐 形式化证明示例

**证明：12n² - 3n = Ω(n²)**

我们需要找到 c、n0，使得：
```
c·n² ≤ 12n² - 3n
```

两边除以 n²（n ≥ 1）：
```
c ≤ 12 - 3/n
```

当 n ≥ 7 时，12 - 3/n ≥ 12 - 3/7 ≈ 11.57

取 **c = 1，n0 = 1**，则对所有 n ≥ 1：
```
n² ≤ 12n² - 3n  ✓
```

---

### o记号（非紧确上界）

#### 📝 定义

```
o(g(n)) = { f(n) : 
    对于任意正常量 c > 0，
    总存在 n0，使得对所有 n ≥ n0，有：
    0 ≤ f(n) ≤ c·g(n)
}
```

#### 🔑 与 O 的关键区别

| 记号 | 条件 | 含义 |
|:---:|:---|:---|
| **O(g(n))** | 存在某个 c > 0 | "f 不超过 g 的某个倍数" |
| **o(g(n))** | 对**任意**c > 0 | "f 远小于 g，比任何比例都小" |

#### 🏠 生活类比：蚂蚁 vs 大象

- 大象体重 = O(蚂蚁体重)：技术上正确（有某个倍数使得成立）
- 大象体重 ≠ o(蚂蚁体重)：错误！因为不可能"远小于"
- 5岁小孩体重 = o(大象体重)：正确！可以找到某个年龄后成立

#### 📐 数学理解

```
f(n) = o(g(n))  ⇔  lim(n→∞) f(n)/g(n) = 0
```

**当 n 趋于无穷时，f(n) 相对于 g(n) 变得微不足道！**

#### 📐 形式化证明示例

**证明：n = o(n²)**

计算极限：
```
lim(n→∞) n/n² = lim(n→∞) 1/n = 0
```

所以 n = o(n²) ✓

**为什么 n ≠ o(n)？**

```
lim(n→∞) n/n = 1 ≠ 0
```

---

### ω记号（非紧确下界）

#### 📝 定义

```
ω(g(n)) = { f(n) : 
    对于任意正常量 c > 0，
    总存在 n0，使得对所有 n ≥ n0，有：
    0 ≤ c·g(n) < f(n)
}
```

#### 🔑 与 Ω 的关键区别

| 记号 | 条件 | 含义 |
|:---:|:---|:---|
| **Ω(g(n))** | 存在某个 c > 0 | "f 不低于 g 的某个倍数" |
| **ω(g(n))** | 对**任意**c > 0 | "f 远大于 g，比任何比例都大" |

#### 🏠 生活类比：长大成人的孩子

- 你小时候 = ω(蚂蚁体重)：错误！太小了
- 你长大后 = ω(小时候的体重)：正确！你永远比小时候重

#### 📐 数学理解

```
f(n) = ω(g(n))  ⇔  lim(n→∞) f(n)/g(n) = ∞
```

**当 n 趋于无穷时，f(n) 相对于 g(n) 变得无穷大！**

---

### 📊 五种记号总结

| 记号 | 读音 | 含义 | 类比 |
|:---:|:---:|:---|:---|
| **Θ(g(n))** | "g(n) 的 Theta" | f(n) **等于** g(n)（同阶） | ≈ |
| **O(g(n))** | "大 O" | f(n) **不超过** g(n)（上界） | ≤ |
| **Ω(g(n))** | "大 Omega" | f(n) **不低于** g(n)（下界） | ≥ |
| **o(g(n))** | "小 o" | f(n) **严格小于** g(n)（非紧上界） | < |
| **ω(g(n))** | "小 omega" | f(n) **严格大于** g(n)（非紧下界） | > |

#### 🎨 图解总览

```
f(n) = ω(g(n))     f(n) >> g(n)    远大于
f(n) = Ω(g(n))     f(n) >= g(n)    至少是
f(n) = Θ(g(n))     f(n) ≈ g(n)     同阶
f(n) = O(g(n))     f(n) <= g(n)    不超过
f(n) = o(g(n))     f(n) << g(n)    远小于
```

## 3.2 渐近记号的性质

### 🔗 传递性

就像数字比较一样，渐近关系可以传递：

| 记号 | 传递性 |
|:---:|:---|
| **Θ** | f(n) = Θ(g(n)) 且 g(n) = Θ(h(n)) ⇒ f(n) = Θ(h(n)) |
| **O** | f(n) = O(g(n)) 且 g(n) = O(h(n)) ⇒ f(n) = O(h(n)) |
| **Ω** | f(n) = Ω(g(n)) 且 g(n) = Ω(h(n)) ⇒ f(n) = Ω(h(n)) |
| **o** | f(n) = o(g(n)) 且 g(n) = o(h(n)) ⇒ f(n) = o(h(n)) |
| **ω** | f(n) = ω(g(n)) 且 g(n) = ω(h(n)) ⇒ f(n) = ω(h(n)) |

**生活类比**：
- 小明比小华高，小华比小李高 → 小明比小李高 ✓

### 🔄 自反性

每个"等于"型记号都和自己相等：

| 记号 | 自反性 |
|:---:|:---|
| **Θ** | f(n) = Θ(f(n)) ✓ |
| **O** | f(n) = O(f(n)) ✓ |
| **Ω** | f(n) = Ω(f(n)) ✓ |

**生活类比**：
- 小明和自己一样高 ✓

### 🔀 对称性

只有 Θ 有对称性：

| 记号 | 对称性 |
|:---:|:---|
| **Θ** | f(n) = Θ(g(n)) ⇔ g(n) = Θ(f(n)) ✓ |

**生活类比**：
- 小明和小华一样高 → 小华和小明一样高 ✓

### ↔ 转置对称性

O 和 Ω、o 和 ω 互为转置对称：

| 记号对 | 转置对称性 |
|:---:|:---|
| **O 和 Ω** | f(n) = O(g(n)) ⇔ g(n) = Ω(f(n)) |
| **o 和 ω** | f(n) = o(g(n)) ⇔ g(n) = ω(f(n)) |

**生活类比**：
- "小明不超过小华" ⇔ "小华不低于小明" ✓

### ⚠️ 注意：三分性不成立

实数有三分性：对于任意两个实数 a 和 b，恰好有 a < b、a = b 或 a > b 之一成立。

**但渐近记号没有三分性！**

#### ❌ 反例：n 和 n^(1+sin n)

考虑 f(n) = n 和 g(n) = n^(1+sin n)：

- 当 sin n = 1 时，g(n) = n²，比 f(n) 大
- 当 sin n = -1 时，g(n) = n⁰ = 1，比 f(n) 小
- 当 sin n = 0 时，g(n) = n，和 f(n) 相等

所以 n 既不是 O(g(n)) 也不是 Ω(g(n))！

**生活类比**：两个人的身高忽上忽下，无法比较。

## 3.3 标准记号与常用函数

### 🔢 向下取整与向上取整

| 记号 | 含义 | Python 示例 |
|:---:|:---|:---|
| ⌊x⌋ | 不超过 x 的最大整数 | `math.floor(x)` |
| ⌈x⌉ | 不小于 x 的最小整数 | `math.ceil(x)` |

#### 📐 重要性质

```
⌊x⌋ ≤ x ≤ ⌈x⌉

x - 1 < ⌊x⌋ ≤ x ≤ ⌈x⌉ < x + 1
```

#### 🏠 生活类比：电影院座位

- 你需要 ⌈人数/座位数⌉ 排
- 3人，10座一排 → ⌈3/10⌉ = 1排
- 25人，10座一排 → ⌈25/10⌉ = 3排

---

### 📐 对数函数

#### 📝 记号约定（算法导论标准）

| 记号 | 含义 |
|:---:|:---|
| **lg n** | log₂n（二进制对数） |
| **ln n** | logₑn（自然对数） |
| **lg⁽ᵏ⁾ n** | (lg n)ᵏ（幂） |
| **lg lg n** | lg(lg n)（复合） |

#### 🏠 生活类比：翻倍与对数

- 翻倍游戏：你有 1 元，每次翻倍
- 1, 2, 4, 8, 16, 32, 64, ...
- 想要 1000 元需要翻多少次？
- **lg₂(1000) ≈ 10 次**（因为 2¹⁰ = 1024）

#### 📐 重要性质

```
lg(nm) = lg n + lg m
lg(n/m) = lg n - lg m
lg nᵏ = k · lg n
lg lg n = lg(ln n)
```

#### ⚠️ 常见错误

**lg⁽²⁾n ≠ lg(lg n)**

- lg⁽²⁾ n = lg(lg(n))（连续两次取对数）
- lg² n = (lg n)²（对数的平方）

---

### 📈 指数函数

#### 📝 记号约定

```
aⁿ = a × a × ... × a (n个a)
```

#### 🏠 生活类比：细胞分裂

- 1个细胞 → 2个 → 4个 → 8个 → ...
- n 次分裂后有 2ⁿ 个细胞
- **指数增长极快！**

#### 📐 重要性质

```
a⁰ = 1
a¹ = a
aⁿ · aᵐ = aⁿ⁺ᵐ
(aⁿ)ᵐ = aⁿᵐ
aⁿ / aᵐ = aⁿ⁻ᵐ
```

#### ⚠️ 指数 vs 多项式

**重要结论**（需记住！）：

```
对任意 a > 1 和 b > 0：
nᵇ = o(aⁿ)

翻译：任何多项式都比任何指数增长慢
```

**验证**：
```
n = 10:    n² = 100,     2ⁿ = 1024
n = 20:    n² = 400,      2ⁿ = 1,048,576
n = 100:   n² = 10,000,   2ⁿ ≈ 1.27×10³⁰
```

---

### 🔢 阶乘函数

#### 📝 定义

```
n! = n × (n-1) × (n-2) × ... × 2 × 1
```

#### 🏠 生活类比：排列顺序

- 3 本书的不同排列：3! = 6 种
- 5 个人排队：5! = 120 种

#### 📐 重要性质（斯德林公式近似）

```
n! = Θ(nⁿ · e⁻ⁿ · √(2πn))  ← 斯德林近似

简化版：n! = o(nⁿ)          ← 阶乘比 nⁿ 增长慢
n! = ω(2ⁿ)                  ← 阶乘比 2ⁿ 增长快
lg(n!) = Θ(n lg n)          ← 常用结论！
```

#### ⚠️ 验证 lg(n!) = Θ(n lg n)

**下界**：
```
lg(n!) = lg(1·2·3·...·n)
       ≥ lg((n/2)·(n/2+1)·...·n)      # 只取后一半
       ≥ lg((n/2)·(n/2)·...·(n/2))    # 每项都 ≥ n/2
       = (n/2) · lg(n/2)              # 共 n/2 个 n/2 相乘
       = Ω(n lg n)
```

**上界**：
```
lg(n!) = lg(1·2·3·...·n)
       ≤ lg(n·n·n·...·n)              # 每项都 ≤ n
       = n · lg n                      # 共 n 个 n 相乘
       = O(n lg n)
```

所以 lg(n!) = Θ(n lg n) ✓

---

### 🔄 多重函数

#### 📝 定义

```
f⁽ⁱ⁾(n) = f(f⁽ⁱ⁻¹⁾(n))      # f 作用 i 次

f⁽⁰⁾(n) = n                  # 0次作用等于自身
```

#### 🏠 生活类比：翻倍再翻倍

- f(n) = 2n（翻倍）
- f⁽²⁾(n) = f(f(n)) = f(2n) = 4n
- f⁽³⁾(n) = f(f(f(n))) = f(4n) = 8n

#### 📐 示例

若 f(n) = 2n，则：
```
f⁽ⁱ⁾(n) = 2ⁱ · n
```

---

### 📊 多重对数函数

#### 📝 定义

```
lg⁽ⁱ⁾ n =                          # 从 n 开始连续取 i 次对数
    n                        若 i = 0
    lg(lg⁽ⁱ⁻¹⁾ n)            若 i ≥ 1 且 lg⁽ⁱ⁻¹⁾ n > 0
```

**注意**：只有当 lg⁽ⁱ⁻¹⁾ n > 0 时才有定义（非正数无对数）

#### 🏠 生活类比：不断翻倍再翻倍的对立面

- 翻倍增长：n → 2n → 4n → 8n → ...
- 多重对数增长：n → lg n → lg lg n → lg lg lg n → ...

**多重对数增长极慢！**

#### 📐 实际意义

```
lg⁽¹⁾ 1000 ≈ 10
lg⁽²⁾ 1000 ≈ lg(10) ≈ 3.3
lg⁽³⁾ 1000 ≈ lg(3.3) ≈ 1.7
lg⁽⁴⁾ 1000 ≈ lg(1.7) ≈ 0.8
```

即使 n = 10¹⁰⁰（宇宙原子数级别），lg⁽⁴⁾ n 也只有约 100！

| **渐近记号的目的** | 用简单函数近似描述复杂函数的增长趋势 |
| **Θ 记号** | 渐近紧确界，上下界都紧 |
| **O 记号** | 渐近上界，最常用 |
| **Ω 记号** | 渐近下界 |
| **o 记号** | 严格小于，非紧确上界 |
| **ω 记号** | 严格大于，非紧确下界 |
| **函数增长顺序** | 常数 < 对数 < 多项式 < 指数 < 阶乘 |

### 📈 常见函数的增长顺序

从慢到快排列：

```
1    <    lg n    <    n    <    n lg n    <    n²    <    n³    <    2ⁿ    <    n!
```

**生活类比**：
- 1：静止不动
- lg n：乌龟爬（很慢但一直在走）
- n：人走路
- n lg n：骑自行车
- n²：开汽车
- n³：坐高铁
- 2ⁿ：坐火箭
- n!：瞬间传送

### ⚠️ 常见误区

1. **O(n²) 不一定是紧确上界**
   - n² = O(n³) ✓（但不是 Θ(n³)）

2. **Θ(n) 蕴含 O(n) 和 Ω(n)**
   - 紧确界比单独的上界或下界更强

3. **常数不重要**
   - f(n) = 1000n 和 g(n) = n 都是 Θ(n)
   - 系数不重要，增长级别才重要

4. **低阶项忽略**
   - 2n² + 3n + 5 = Θ(n²)
   - 保留最高阶项，忽略其他

## 思考题与证明

### 证明题1：证明 Θ 记号的定义

**题目**：证明若 f(n) = Θ(g(n))，则 g(n) = Θ(f(n))

**解答**：

由 f(n) = Θ(g(n))，存在 c₁、c₂、n₀ 使得：
```
c₁·g(n) ≤ f(n) ≤ c₂·g(n)    (对所有 n ≥ n₀)
```

两边除以 c₁：
```
g(n) ≤ (1/c₁)·f(n)
```

两边除以 c₂：
```
(1/c₂)·f(n) ≤ g(n)
```

综合得：
```
(1/c₂)·f(n) ≤ g(n) ≤ (1/c₁)·f(n)
```

令 c₁' = 1/c₂，c₂' = 1/c₁，则：
```
c₁'·f(n) ≤ g(n) ≤ c₂'·f(n)
```

即 g(n) = Θ(f(n)) ✓

---

### 证明题2：lg(n!) = Θ(n lg n)

**题目**：使用斯特林近似或夹逼定理证明

**解答**（夹逼定理）：

**上界**：
```
lg(n!) = lg(1·2·3·...·n)
       ≤ lg(n·n·...·n)      # 每个因子 ≤ n
       = n·lg n
       = O(n lg n)
```

**下界**：
```
lg(n!) = lg(1·2·3·...·n)
       ≥ lg((n/2)·(n/2+1)·...·n)      # 只取后一半
       ≥ lg((n/2)·(n/2)·...·(n/2))   # 共 n/2 项，每项 ≥ n/2
       = (n/2)·lg(n/2)
       = (n/2)·(lg n - 1)
       = Ω(n lg n)
```

由上界和下界得：
```
lg(n!) = Θ(n lg n) ✓
```

---

### 选择题：判断下列说法的正确性

1. **2n² + 3n + 1 = O(n³)** ✓（非紧确上界）

2. **2n² + 3n + 1 = Θ(n³)** ✗（应该是 Θ(n²)）

3. **n = Ω(lg n)** ✓（n 增长比 lg n 快）

4. **n = o(n lg n)** ✓（严格小于）

5. **lg n = O(√n)** ✓（对数比平方根增长慢）

## 附录：常用恒等式

### 对数恒等式

```
logₐ(bc) = logₐb + logₐc
logₐ(b/c) = logₐb - logₐc
logₐ(bⁿ) = n·logₐb
logᵦa = logₜa / logₜb   (换底公式)
```

### 指数恒等式

```
a⁰ = 1
a¹ = a
aⁿ · aᵐ = aⁿ⁺ᵐ
(aⁿ)ᵐ = aⁿᵐ
aⁿ / aᵐ = aⁿ⁻ᵐ
aⁿ · bⁿ = (ab)ⁿ
```

### 取整恒等式

```
⌊⌊x⌋⌋ = ⌊x⌋
⌈⌈x⌉⌉ = ⌈x⌉
⌊x⌋ + ⌈x⌉ = 2⌊x⌋ + 1  或  2⌈x⌉ - 1
⌊x⌋ ≤ x ≤ ⌈x⌉
⌊x⌋ = n  ⇔  n ≤ x < n+1
⌈x⌉ = n  ⇔  n-1 < x ≤ n
```
