## 10.1 栈和队列（Stacks and Queues）

### 📖 什么是栈？

**栈**是一种**后进先出（LIFO, Last-In-First-Out）**的数据结构。

```
┌─────────────────────────────────────────┐
│  栈就像一叠盘子：                        │
│                                         │
│    ┌───┐  ← 最后放上去的，最先拿走       │
│    │ 5 │                                │
│    ├───┤                                │
│    │ 4 │                                │
│    ├───┤                                │
│    │ 3 │                                │
│    ├───┤                                │
│    │ 2 │                                │
│    ├───┤                                │
│    │ 1 │  ← 最先放上去的，最后拿走        │
│    └───┘                                │
│                                         │
│  只能在栈顶操作：放入叫 push，拿走叫 pop │
└─────────────────────────────────────────┘
```

### 🏠 生活类比：自助餐厅的盘子

想象自助餐厅的盘子架：

```
初始状态：    放入盘子1：   放入盘子2：   取出盘子（pop）：

┌─────┐       ┌─────┐       ┌─────┐       ┌─────┐
│     │       │     │       │  2  │  ←栈顶  │     │  ← 拿走盘子2
├─────┤       ├─────┤       ├─────┤       ├─────┤
│     │       │  1  │  ←栈顶  │  1  │       │  1  │
├─────┤       ├─────┤       ├─────┤       ├─────┤
│     │       │     │       │     │       │     │
└─────┘       └─────┘       └─────┘       └─────┘

操作规则：
- push：只能从顶部放
- pop：只能从顶部拿
- peek/top：只看不拿
```

### 💡 栈的基本操作

| 操作 | 作用 | 时间复杂度 |
|:--:|:--|:--:|
| **push(s, x)** | 将元素 x 压入栈 s | O(1) |
| **pop(s)** | 弹出栈顶元素并返回 | O(1) |
| **top(s)** | 返回栈顶元素（不弹出） | O(1) |
| **empty(s)** | 判断栈是否为空 | O(1) |

### 📊 过程图解：栈操作

```
push 操作：
        
Stack: [1, 2, 3]      push(4)
                        ↓
                [1, 2, 3, 4]  ← 新元素在栈顶

pop 操作：
        
Stack: [1, 2, 3, 4]    pop() → 返回 4
                        ↓
                [1, 2, 3]      ← 4 被移除

stack overflow / underflow：
- overflow：栈满了还往里塞（数组实现时）
- underflow：栈空了还往外拿
```

### 🐍 Python 实现（逐行注释）

```python
class Stack:
    """
    基于 Python 列表实现的栈
    
    核心原理：用列表的 append() 和 pop() 实现栈的 push 和 pop
    - append() 在列表末尾添加元素 → push
    - pop() 移除并返回列表末尾元素 → pop
    """
    
    def __init__(self):
        """
        初始化栈
        使用空列表存储元素
        """
        self.items = []  # 存储栈元素的列表
    
    def push(self, x):
        """
        将元素压入栈顶
        
        参数:
            x: 要压入的元素
        
        原理:
            append() 在列表末尾添加元素
            这是 O(1) 操作，因为列表末尾添加是常量时间
        """
        self.items.append(x)  # 等价于 self.items[len(self.items):] = [x]
    
    def pop(self):
        """
        弹出栈顶元素
        
        返回:
            栈顶元素
        
        原理:
            pop() 移除并返回列表最后一个元素
            这是 O(1) 操作
            
        注意:
            如果栈为空，会抛出 IndexError
        """
        if self.empty():
            raise IndexError("Stack underflow: cannot pop from empty stack")
        return self.items.pop()  # 默认 pop() 移除并返回最后一个元素
    
    def top(self):
        """
        返回栈顶元素（不弹出）
        
        返回:
            栈顶元素
        
        原理:
            用负索引 self.items[-1] 获取最后一个元素
            这是 O(1) 操作
            
        注意:
            如果栈为空，会抛出 IndexError
        """
        if self.empty():
            raise IndexError("Stack empty: cannot get top of empty stack")
        return self.items[-1]  # -1 表示最后一个元素
    
    def empty(self):
        """
        判断栈是否为空
        
        返回:
            bool: 栈为空返回 True，否则返回 False
        """
        return len(self.items) == 0  # len() 获取列表长度，O(1)
    
    def size(self):
        """
        返回栈中元素个数
        
        返回:
            int: 元素个数
        """
        return len(self.items)


# 测试代码
if __name__ == "__main__":
    s = Stack()
    
    # 测试 push
    s.push(1)
    s.push(2)
    s.push(3)
    print(f"Push后栈: {s.items}")  # [1, 2, 3]
    
    # 测试 top
    print(f"栈顶元素: {s.top()}")  # 3
    
    # 测试 pop
    print(f"Pop返回: {s.pop()}")   # 3
    print(f"Pop后栈: {s.items}")   # [1, 2]
    
    # 测试 empty
    print(f"栈是否为空: {s.empty()}")  # False
```

### 🖥️ C++ 实现（逐行注释）

```cpp
#include <iostream>
#include <vector>
#include <stdexcept>

template <typename T>
class Stack {
    /*
     * 基于 vector 实现的栈
     * 
     * vector 是动态数组，封装了动态内存管理
     * - push_back(): 在末尾添加元素
     * - pop_back(): 移除末尾元素
     * - back(): 获取末尾元素
     * - empty(): 判断是否为空
     */
private:
    std::vector<T> data;  // 存储栈元素的动态数组

public:
    /**
     * push() - 将元素压入栈顶
     * 
     * @param x: 要压入的元素
     * 
     * 原理:
     * push_back() 在 vector 末尾添加元素
     * 平均时间复杂度 O(1)，偶尔可能触发扩容（O(n)）
     */
    void push(const T& x) {
        data.push_back(x);  // vector 的 push_back 操作
    }
    
    /**
     * pop() - 弹出栈顶元素
     * 
     * @return: 被弹出的栈顶元素
     * 
     * 注意:
     * - 栈为空时抛出异常
     * - 不会返回被弹出元素的值（如果需要，用 top() 先获取）
     */
    void pop() {
        if (empty()) {
            throw std::out_of_range("Stack underflow: cannot pop from empty stack");
        }
        data.pop_back();  // 移除 vector 末尾元素
    }
    
    /**
     * top() - 返回栈顶元素（不弹出）
     * 
     * @return: 栈顶元素的引用
     * 
     * 原理:
     * vector::back() 返回最后一个元素的引用
     * 返回引用避免拷贝，提高效率
     */
    T& top() {
        if (empty()) {
            throw std::out_of_range("Stack empty: cannot get top of empty stack");
        }
        return data.back();  // 返回最后一个元素的引用
    }
    
    /**
     * top() 的 const 版本 - 常对象调用
     */
    const T& top() const {
        if (empty()) {
            throw std::out_of_range("Stack empty: cannot get top of empty stack");
        }
        return data.back();
    }
    
    /**
     * empty() - 判断栈是否为空
     * 
     * @return: 栈为空返回 true，否则返回 false
     */
    bool empty() const {
        return data.empty();  // vector::empty() 是 O(1)
    }
    
    /**
     * size() - 返回栈中元素个数
     * 
     * @return: 元素个数
     */
    size_t size() const {
        return data.size();  // vector::size() 是 O(1)
    }
};

// 测试代码
int main() {
    Stack<int> s;
    
    // 测试 push
    s.push(1);
    s.push(2);
    s.push(3);
    std::cout << "Push后栈顶: " << s.top() << std::endl;  // 3
    
    // 测试 pop
    std::cout << "Pop返回: " << s.top() << std::endl;  // 3
    s.pop();
    std::cout << "Pop后栈顶: " << s.top() << std::endl;  // 2
    
    // 测试 empty
    std::cout << "栈是否为空: " << (s.empty() ? "是" : "否") << std::endl;  // 否
    
    return 0;
}
```

### ❓ FAQ：栈有什么用？

1. **函数调用**：函数调用时把返回地址压栈，返回时弹出——这就是递归的原理
2. **撤销操作**：每次操作记录到栈，撤销时弹出
3. **表达式求值**：中缀转后缀、后缀表达式求值
4. **括号匹配**：左括号压栈，右括号匹配弹出
5. **浏览器前进/后退**：前进后退按钮就是个栈

### 📖 什么是队列？

**队列**是一种**先进先出（FIFO, First-In-First-Out）**的数据结构。

```
┌─────────────────────────────────────────┐
│  队列就像排队买奶茶：                     │
│                                         │
│  入队(enqueue) → [1, 2, 3, 4, 5] → 出队(dequeue)
│                  ↑              ↑
│                队首            队尾       │
│              (front)          (rear)    │
│                                         │
│  先来的先买，后来的排队                  │
└─────────────────────────────────────────┘
```

### 🏠 生活类比：奶茶店排队

```
初始队列：    队首        队尾
            [小明] [小红] [小华]
              ↑           ↑
            front        rear

小绿入队后：  [小明] [小红] [小华] [小绿]
              ↑                       ↑
            front                   rear

小明出队后：  [小红] [小华] [小绿]
              ↑               ↑
            front            rear

规则：
- 入队(enqueue)：在队尾添加
- 出队(dequeue)：在队尾移除（队首）
```

### 💡 队列的基本操作

| 操作 | 作用 | 时间复杂度 |
|:--:|:--|:--:|
| **enqueue(Q, x)** | 将元素 x 加入队尾 | O(1) |
| **dequeue(Q)** | 移除并返回队首元素 | O(1) |
| **front(Q)** | 返回队首元素（不删除） | O(1) |
| **empty(Q)** | 判断队列是否为空 | O(1) |

### 📊 过程图解：循环队列

```
普通队列的问题：出队后空间浪费

初始：      出队2次后：
[1, 2, 3, 4, 5, _, _, _, _]
  ↑                   ↑
 front               rear

解决方案：循环队列（环形缓冲区）

用数组实现，队首队尾指针环绕：

状态1：        状态2（入队4）：   状态3（出队2次）：
┌───┬───┬───┬───┐  ┌───┬───┬───┬───┐  ┌───┬───┬───┬───┐
│ 3 │ 4 │   │   │  │ 3 │ 4 │ 1 │   │  │ 3 │ 4 │ 1 │ 2 │
├───┼───┼───┼───┤  ├───┼───┼───┼───┤  ├───┼───┼───┼───┤
  ↑           ↑      ↑       ↑          ↑   ↑
 rear       front   front    rear       rear  front

队首指针指向队首元素
队尾指针指向下一个空位（或最后一个元素，视实现而定）
```

### 🐍 Python 实现（逐行注释）

```python
class Queue:
    """
    基于列表实现的队列（简单版，非循环）
    
    简单实现：
    - 入队用 append() 在末尾添加
    - 出队用 pop(0) 移除第一个元素
    
    注意：pop(0) 是 O(n) 操作，因为要移动所有元素
    如果追求效率，可以用 collections.deque
    """
    
    def __init__(self):
        """初始化空队列"""
        self.items = []
    
    def enqueue(self, x):
        """
        入队：将元素加入队尾
        
        参数:
            x: 要加入的元素
        
        原理:
            append() 在列表末尾添加元素
            这是 O(1) 操作
        """
        self.items.append(x)
    
    def dequeue(self):
        """
        出队：移除并返回队首元素
        
        返回:
            队首元素
        
        注意:
            - 如果队列为空，抛出 IndexError
            - pop(0) 是 O(n) 操作，因为要移动所有元素
        
        优化方案：
            使用 collections.deque，其 popleft() 是 O(1)
        """
        if self.empty():
            raise IndexError("Queue underflow: cannot dequeue from empty queue")
        return self.items.pop(0)  # 移除并返回第一个元素
    
    def front(self):
        """
        返回队首元素（不删除）
        
        返回:
            队首元素
        """
        if self.empty():
            raise IndexError("Queue empty: cannot get front of empty queue")
        return self.items[0]
    
    def empty(self):
        """判断队列是否为空"""
        return len(self.items) == 0
    
    def size(self):
        """返回队列中元素个数"""
        return len(self.items)


class EfficientQueue:
    """
    高效队列：使用 collections.deque
    
    deque（双端队列）的特点：
    - 两端添加/删除都是 O(1)
    - 比 list.pop(0) 高效得多
    """
    
    def __init__(self):
        """
        初始化队列
        使用 deque 而非 list
        """
        from collections import deque
        self.items = deque()  # deque 是双端队列
    
    def enqueue(self, x):
        """
        入队：在队尾添加元素
        
        原理:
            append() 在 deque 末尾添加，O(1)
        """
        self.items.append(x)  # O(1)
    
    def dequeue(self):
        """
        出队：移除并返回队首元素
        
        原理:
            popleft() 从 deque 左边移除，O(1)
            比 list.pop(0) 高效得多
        """
        if self.empty():
            raise IndexError("Queue underflow: cannot dequeue from empty queue")
        return self.items.popleft()  # O(1)!
    
    def front(self):
        """返回队首元素"""
        if self.empty():
            raise IndexError("Queue empty: cannot get front of empty queue")
        return self.items[0]
    
    def empty(self):
        """判断队列是否为空"""
        return len(self.items) == 0


# 测试代码
if __name__ == "__main__":
    q = EfficientQueue()
    
    # 测试入队
    q.enqueue("小明")
    q.enqueue("小红")
    q.enqueue("小华")
    print(f"入队后队首: {q.front()}")  # 小明
    
    # 测试出队
    print(f"出队: {q.dequeue()}")   # 小明
    print(f"出队: {q.dequeue()}")   # 小红
    print(f"现在队首: {q.front()}")  # 小华
```

### 🖥️ C++ 实现（逐行注释）

```cpp
#include <iostream>
#include <queue>  // 标准库提供了 queue

/*
 * C++ 标准库 queue 容器
 * 
 * 基于 deque 或 list 实现，提供了高效的队列操作：
 * - push(): 入队
 * - pop(): 出队（不返回值）
 * - front(): 获取队首
 * - back(): 获取队尾
 * - empty(): 判断是否为空
 * - size(): 获取元素个数
 */

// 如果要自己实现循环队列：
template <typename T>
class CircularQueue {
    /*
     * 循环队列实现
     * 
     * 使用固定大小的数组，通过取模运算实现环绕
     * 优点：不浪费空间
     * 缺点：队列满时需要特殊处理（或者牺牲一个空间）
     */
private:
    T* data;           // 存储数据的数组
    int capacity;      // 队列容量
    int front_idx;    // 队首索引
    int rear_idx;     // 队尾索引（下一个入队位置）

public:
    CircularQueue(int cap = 10) : capacity(cap), front_idx(0), rear_idx(0) {
        /*
         * 构造函数
         * 
         * 初始化：front_idx = rear_idx = 0 表示空队列
         */
        data = new T[capacity];
    }
    
    ~CircularQueue() {
        delete[] data;
    }
    
    bool enqueue(const T& x) {
        /*
         * 入队
         * 
         * @param x: 要入队的元素
         * @return: 入队成功返回 true，队列满返回 false
         * 
         * 判断队列满：
         *   (rear + 1) % capacity == front
         *   牺牲一个空间来区分队满和队空
         */
        if (isFull()) {
            return false;  // 队列满，入队失败
        }
        data[rear_idx] = x;
        rear_idx = (rear_idx + 1) % capacity;  // 循环移动
        return true;
    }
    
    bool dequeue(T& x) {
        /*
         * 出队
         * 
         * @param x: 用于存储出队元素
         * @return: 出队成功返回 true，队列空返回 false
         */
        if (isEmpty()) {
            return false;  // 队列空
        }
        x = data[front_idx];
        front_idx = (front_idx + 1) % capacity;  // 循环移动
        return true;
    }
    
    bool front(T& x) {
        /* 获取队首元素 */
        if (isEmpty()) {
            return false;
        }
        x = data[front_idx];
        return true;
    }
    
    bool isEmpty() {
        /*
         * 判断队列是否为空
         * 
         * 队空条件：front_idx == rear_idx
         */
        return front_idx == rear_idx;
    }
    
    bool isFull() {
        /*
         * 判断队列是否已满
         * 
         * 队满条件：(rear_idx + 1) % capacity == front_idx
         * 因为牺牲了一个空间来区分队满和队空
         */
        return (rear_idx + 1) % capacity == front_idx;
    }
};

// 测试代码
int main() {
    // 使用标准库 queue
    std::queue<int> q;
    
    q.push(1);
    q.push(2);
    q.push(3);
    
    std::cout << "队首: " << q.front() << std::endl;  // 1
    
    q.pop();
    std::cout << "出队后队首: " << q.front() << std::endl;  // 2
    
    // 使用自定义循环队列
    CircularQueue<int> cq(5);
    
    for (int i = 1; i <= 4; i++) {
        cq.enqueue(i);
    }
    
    int x;
    cq.dequeue(x);
    std::cout << "循环队列出队: " << x << std::endl;
    
    return 0;
}
```

### ❓ FAQ：队列有什么用？

1. **任务调度**：操作系统按顺序执行任务
2. **广度优先搜索（BFS）**：图的遍历
3. **打印队列**：多个人同时打印，排队处理

## 10.2 链表（Linked Lists）

### 📖 什么是链表？

**链表**是一种线性数据结构，数据元素通过**指针**连接在一起。

```
┌─────────────────────────────────────────┐
│  链表就像寻宝游戏：                       │
│                                         │
│  每张纸条写着一个数字 + 下一张纸条的位置   │
│                                         │
│  ┌────┬────┐    ┌────┬────┐    ┌────┬────┐
│  │ 10 │ ●──┼──→ │ 20 │ ●──┼──→ │ 30 │ NULL│
│  └────┴────┘    └────┴────┘    └────┴────┘
│     节点0        节点1         节点2
│                                         │
│  ● = 指向下一个节点的指针                 │
│  NULL = 链表结束标记                     │
└─────────────────────────────────────────┘
```

### 🏠 生活类比：寻宝游戏

想象你和小伙伴玩寻宝游戏：

```

这就是链表：每个节点有数据 + 下一个节点的位置
```

### 💡 链表的类型

```
1. 单向链表（Singly Linked List）
   
   head → [A] → [B] → [C] → NULL
   只能从头到尾遍历
   
2. 双向链表（Doubly Linked List）
   
   NULL ← → [A] ← → [B] ← → [C] ← → NULL
   可以从头到尾，也可以从尾到头
   
3. 循环链表（Circular Linked List）
   
   head → [A] → [B] → [C] ─┐
          ↑                 │
          └─────────────────┘
   最后一个节点的指针指向头节点，形成环
```

class Node:
    """
    链表节点类
    
    包含两个属性：
    - key: 节点存储的数据
    - next: 指向下一个节点的引用（None 表示链表末尾）
    """
    
    def __init__(self, key=None):
        """
        初始化节点
        
        参数:
            key: 节点存储的数据，默认为 None
        """
        self.key = key  # 节点的数据
        self.next = None  # 指向下一个节点的引用


class LinkedList:
    """
    单向链表实现
    
    包含一个 head 指针指向第一个节点
    
    基本操作：
    - insert: 在头部插入（O(1)）
    - delete: 删除指定节点（需要先找到）
    - search: 搜索指定值的节点（O(n)）
    """
    
    def __init__(self):
        """
        初始化空链表
        空链表的 head 为 None
        """
        self.head = None  # 头节点，None 表示空链表
    
    def insert(self, key):
        """
        在链表头部插入新节点
        
        参数:
            key: 要插入的数据
        
        原理（O(1)）：
        1. 创建新节点，新节点的 next 指向当前的 head
        2. 将 head 指向新节点
        
        为什么是 O(1)？
        - 只需要改变两个指针
        - 不需要遍历链表
        """
        new_node = Node(key)  # 1. 创建新节点
        new_node.next = self.head  # 2. 新节点的 next 指向原 head
        self.head = new_node  # 3. head 指向新节点
    
    def search(self, key):
        """
        搜索值为 key 的节点
        
        参数:
            key: 要搜索的值
        
        返回:
            找到返回节点，未找到返回 None
        
        时间复杂度: O(n) - 最坏情况要遍历整个链表
        """
        current = self.head  # 从头节点开始
        while current is not None:  # 遍历直到链表末尾
            if current.key == key:
                return current  # 找到目标
            current = current.next  # 移动到下一个节点
        return None  # 未找到
    
    def delete(self, key):
        """
        删除第一个值为 key 的节点
        
        参数:
            key: 要删除的值
        
        返回:
            删除成功返回 True，未找到返回 False
        
        时间复杂度: O(n) - 需要先遍历找到节点
        """
        # 处理 head 是要删除节点的情况
        if self.head is not None and self.head.key == key:
            self.head = self.head.next
            return True
        
        # 遍历找到目标节点及其前驱
        prev = self.head
        current = self.head.next if self.head else None
        
        while current is not None:
            if current.key == key:
                # 跳过目标节点
                prev.next = current.next
                return True
            prev = current
            current = current.next
        
        return False  # 未找到
    
    def traverse(self):
        """
        遍历链表，输出所有节点的值
        
        返回:
            list: 包含所有节点值的列表
        """
        result = []
        current = self.head
        while current is not None:
            result.append(current.key)
            current = current.next
        return result
    
    def __str__(self):
        """打印链表的字符串表示"""
        return " → ".join(str(k) for k in self.traverse()) + " → NULL"
    
    def empty(self):
        """判断链表是否为空"""
        return self.head is None

#include <iostream>

/**
 * 链表节点结构
 * 
 * C++ 中用 struct 定义简洁的类
 * - key: 节点存储的数据
 * - next: 指向下一个节点的指针
 */
struct Node {
    int key;        // 节点数据
    Node* next;     // 指向下一个节点的指针
    
    // 构造函数
    Node(int k = 0, Node* n = nullptr) : key(k), next(n) {}
};

/**
 * 单链表类
 */
class LinkedList {
private:
    Node* head;  // 头节点指针

public:
    /**
     * 构造函数
     * 
     * 初始化：head = nullptr 表示空链表
     */
    LinkedList() : head(nullptr) {}
    
    /**
     * 析构函数
     * 
     * 释放所有节点的内存，防止内存泄漏
     */
    ~LinkedList() {
        Node* current = head;
        while (current != nullptr) {
            Node* next = current->next;  // 先保存下一个节点
            delete current;  // 释放当前节点
            current = next;
        }
    }
    
    /**
     * insert() - 在链表头部插入
     * 
     * 时间复杂度: O(1)
     * 
     * 步骤:
     * 1. 创建新节点
     * 2. 新节点的 next 指向当前的 head
     * 3. head 指向新节点
     */
    void insert(int key) {
        Node* new_node = new Node(key);  // 1. 创建新节点
        new_node->next = head;  // 2. 新节点指向原头节点
        head = new_node;  // 3. 更新头指针
    }
    
    /**
     * search() - 搜索值为 key 的节点
     * 
     * 时间复杂度: O(n)
     */
    Node* search(int key) {
        Node* current = head;  // 从头节点开始
        
        while (current != nullptr) {  // 遍历直到末尾
            if (current->key == key) {
                return current;  // 找到目标
            }
            current = current->next;  // 移动到下一个
        }
        
        return nullptr;  // 未找到
    }
    
    /**
     * delete_key() - 删除第一个值为 key 的节点
     * 
     * 时间复杂度: O(n)
     * 
     * 特殊情况:
     * - 空链表
     * - 删除的是头节点
     */
    bool delete_key(int key) {
        // 空链表
        if (head == nullptr) {
            return false;
        }
        
        // 删除的是头节点
        if (head->key == key) {
            Node* temp = head;
            head = head->next;
            delete temp;
            return true;
        }
        
        // 遍历找到目标节点及其前驱
        Node* prev = head;
        Node* current = head->next;
        
        while (current != nullptr) {
            if (current->key == key) {
                // 跳过目标节点
                prev->next = current->next;
                delete current;
                return true;
            }
            prev = current;
            current = current->next;
        }
        
        return false;  // 未找到
    }
    
    /**
     * traverse() - 遍历链表
     */
    void traverse() {
        std::cout << "链表: ";
        Node* current = head;
        
        while (current != nullptr) {
            std::cout << current->key;
            if (current->next != nullptr) {
                std::cout << " → ";
            }
            current = current->next;
        }
        std::cout << " → NULL" << std::endl;
    }
    
    bool empty() {
        return head == nullptr;
    }
};

### 💡 双向链表实现

```python
class DNode:
    """双向链表节点"""
    def __init__(self, key=None):
        self.key = key
        self.prev = None  # 指向前一个节点
        self.next = None  # 指向后一个节点


class DoublyLinkedList:
    """
    双向链表
    
    特点：
    - 每个节点有 prev 和 next 两个指针
    - 可以双向遍历
    - 删除节点是 O(1)（如果已知节点引用）
    """
    
    def __init__(self):
        self.head = None  # 头节点
        self.tail = None  # 尾节点（方便操作）
    
    def insert_front(self, key):
        """在头部插入"""
        new_node = DNode(key)
        new_node.next = self.head
        
        if self.head:
            self.head.prev = new_node
        else:
            self.tail = new_node  # 空链表时，tail 也是新节点
        
        self.head = new_node
    
    def insert_back(self, key):
        """在尾部插入"""
        new_node = DNode(key)
        new_node.prev = self.tail
        
        if self.tail:
            self.tail.next = new_node
        else:
            self.head = new_node  # 空链表时，head 也是新节点
        
        self.tail = new_node
    
    def delete(self, node):
        """
        删除指定节点（O(1)）
        
        参数:
            node: 要删除的节点引用
        
        优势：
        - 已知节点引用时，删除是 O(1)
        - 不需要遍历找前驱
        """
        if node.prev:
            node.prev.next = node.next
        else:
            self.head = node.next  # 删除的是头节点
        
        if node.next:
            node.next.prev = node.prev
        else:
            self.tail = node.prev  # 删除的是尾节点
```

### ❓ FAQ：链表 vs 数组，哪个更好？

| 特性 | 数组 | 链表 |
|:--|:--|:--|
| 随机访问 | O(1)，通过索引直接访问 | O(n)，需要遍历 |
| 插入/删除（已知位置） | O(n)，需要移动元素 | O(1)，只需改变指针 |
| 内存分配 | 连续，固定大小 | 分散，动态大小 |
| 内存效率 | 高（无额外开销） | 低（需要存储指针） |
| 缓存友好性 | 好（连续内存） | 差（可能跳来跳去） |

**选择建议**：
- 需要频繁随机访问 → 数组
- 需要频繁插入/删除 → 链表
- 内存固定、大小已知 → 数组
- 大小动态变化 → 链表

## 10.3 有根树（Rooted Trees）

### 📖 什么是树？

**树**是一种层级数据结构，由**节点**组成，节点之间有**父子关系**。

```
┌─────────────────────────────────────────┐
│  树就像公司的组织结构图：                 │
│                                         │
│                    👔 CEO                │
│                   /  │  \               │
│                 VP1 VP2 VP3            │
│                / │  \  │               │
│              M1 M2 M3 M4 M5            │
│                                         │
│  特点：                                 │
│  - 只有一个根（CEO）                    │
│  - 每个节点只有一个父节点                │
│  - 没有父节点的叫根节点                  │
│  - 没有子节点的叫叶子节点                │
└─────────────────────────────────────────┘
```

### 🏠 生活类比：家谱

```
中国传统家族树：

        祖父
       / | \
     大伯 父亲 小叔
     /   / \    \
   堂哥  我  弟弟  堂妹

- 根节点：祖父（最老的祖先）
- 父节点：父亲
- 子节点：我、弟弟
- 叶子节点：堂哥、堂妹（没有孩子）
- 深度：距离根节点的层数
- 高度：子树中最长路径的节点数
```

### 💡 树的表示方法

#### 方法1：父节点表示法

```
每个节点只存储父节点的指针

        A
       / \
      B   C          节点:  A  B  C  D  E  F
     / \   \         父节点: -  A  A  B  B  C
    D   E   F        (- 表示根节点)
    
优点：简单，省空间
缺点：找子节点困难
```

#### 方法2：左孩子右兄弟表示法

```
每个节点有：
- left_child: 左子节点
- right_sibling: 右兄弟

        A
       / \
      B   C
     / \   \
    D   E   F

A: left=A.left_child=A, right_sibling=NULL
B: left=D, right_sibling=C
C: left=F, right_sibling=NULL
D: left=NULL, right_sibling=E
E: left=NULL, right_sibling=NULL
F: left=NULL, right_sibling=NULL

优点：只用两个指针就能表示任意树
缺点：操作稍复杂
```

#### 方法3：孩子数组表示法

```
每个节点有多个子节点指针

        A
       /|\
      B C D
     
A: children = [B, C, D]
B: children = []
C: children = []
D: children = []

优点：访问子节点方便
缺点：需要预知最大度（子节点数）
```

### 📊 二叉树图解

```
二叉树：每个节点最多两个子节点

                    A (根)
                   / \
                  B   C
                 / \   \
                D   E   F
               /
              G

用数组表示（层序编号）：
索引:    0   1   2   3   4   5   6
值:     [A,  B,  C,  D,  E,  F,  G]

索引关系：
- 节点 i 的父节点：(i-1) / 2（向下取整）
- 节点 i 的左子：2*i + 1
- 节点 i 的右子：2*i + 2

例如节点 D（索引3）：
- 父节点：(3-1)/2 = 1 → B ✓
- 左子：2*3+1 = 7 → 无（叶子节点）
- 右子：2*3+2 = 8 → 无
```

### 🐍 Python 实现：二叉树节点

```python
class TreeNode:
    """
    二叉树节点
    
    属性：
    - key: 节点的值
    - left: 左子节点
    - right: 右子节点
    - parent: 父节点（可选，方便某些操作）
    """
    
    def __init__(self, key, left=None, right=None, parent=None):
        self.key = key
        self.left = left
        self.right = right
        self.parent = parent
    
    def __str__(self):
        return str(self.key)


class BinaryTree:
    """
    二叉树
    
    基本操作：
    - insert: 插入节点
    - search: 搜索节点
    - traverse: 遍历
    """
    
    def __init__(self):
        self.root = None  # 根节点
    
    def insert(self, key):
        """
        插入节点（构造完全二叉树）
        
        简单实现：层序遍历找到第一个空位插入
        """
        new_node = TreeNode(key)
        
        if self.root is None:
            self.root = new_node
            return
        
        # 层序遍历找空位
        from collections import deque
        queue = deque([self.root])
        
        while queue:
            node = queue.popleft()
            
            # 左子为空，插入到左边
            if node.left is None:
                node.left = new_node
                new_node.parent = node
                return
            else:
                queue.append(node.left)
            
            # 右子为空，插入到右边
            if node.right is None:
                node.right = new_node
                new_node.parent = node
                return
            else:
                queue.append(node.right)
    
    def search(self, key):
        """搜索值为 key 的节点"""
        if self.root is None:
            return None
        
        from collections import deque
        queue = deque([self.root])
        
        while queue:
            node = queue.popleft()
            
            if node.key == key:
                return node
            
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        
        return None
    
    def inorder_traverse(self, node):
        """
        中序遍历：左 → 根 → 右
        
        对于二叉搜索树，结果是升序的
        """
        result = []
        
        def _traverse(n):
            if n is not None:
                _traverse(n.left)
                result.append(n.key)
                _traverse(n.right)
        
        _traverse(node or self.root)
        return result
    
    def preorder_traverse(self, node):
        """
        前序遍历：根 → 左 → 右
        
        用于复制树、计算高度等
        """
        result = []
        
        def _traverse(n):
            if n is not None:
                result.append(n.key)
                _traverse(n.left)
                _traverse(n.right)
        
        _traverse(node or self.root)
        return result
    
    def postorder_traverse(self, node):
        """
        后序遍历：左 → 右 → 根
        
        用于删除树、计算目录大小等
        """
        result = []
        
        def _traverse(n):
            if n is not None:
                _traverse(n.left)
                _traverse(n.right)
                result.append(n.key)
        
        _traverse(node or self.root)
        return result
    
    def height(self, node=None):
        """
        计算树的高度
        
        树的高度 = max(左子树高度, 右子树高度) + 1
        空树高度 = -1
        只有根节点高度 = 0
        """
        node = node or self.root
        
        if node is None:
            return -1
        
        left_height = self.height(node.left)
        right_height = self.height(node.right)
        
        return max(left_height, right_height) + 1


# 测试代码
if __name__ == "__main__":
    bt = BinaryTree()
    
    # 插入节点
    for key in ['A', 'B', 'C', 'D', 'E', 'F', 'G']:
        bt.insert(key)
    
    print(f"树高度: {bt.height()}")  # 2
    
    print(f"前序遍历: {bt.preorder_traverse()}")  # A B D G E C F
    print(f"中序遍历: {bt.inorder_traverse()}")   # D B G E A C F
    print(f"后序遍历: {bt.postorder_traverse()}") # G D E B F C A
```

### 🖥️ C++ 实现：二叉树节点

```cpp
#include <iostream>
#include <queue>

/**
 * 二叉树节点结构
 */
struct TreeNode {
    int key;              // 节点值
    TreeNode* left;       // 左子节点
    TreeNode* right;      // 右子节点
    TreeNode* parent;     // 父节点
    
    TreeNode(int k, TreeNode* p = nullptr, 
             TreeNode* l = nullptr, TreeNode* r = nullptr)
        : key(k), parent(p), left(l), right(r) {}
};

/**
 * 二叉树类
 */
class BinaryTree {
private:
    TreeNode* root;  // 根节点
    
    // 辅助函数：计算高度
    int heightHelper(TreeNode* node) {
        if (node == nullptr) {
            return -1;  // 空树高度 -1
        }
        int left_h = heightHelper(node->left);
        int right_h = heightHelper(node->right);
        return std::max(left_h, right_h) + 1;
    }
    
public:
    BinaryTree() : root(nullptr) {}
    
    /**
     * insert() - 插入节点（完全二叉树）
     * 
     * 时间复杂度: O(n) - 需要遍历找到空位
     */
    void insert(int key) {
        TreeNode* new_node = new TreeNode(key);
        
        if (root == nullptr) {
            root = new_node;
            return;
        }
        
        // 层序遍历找空位
        std::queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            TreeNode* node = q.front();
            q.pop();
            
            // 左子为空
            if (node->left == nullptr) {
                node->left = new_node;
                new_node->parent = node;
                return;
            } else {
                q.push(node->left);
            }
            
            // 右子为空
            if (node->right == nullptr) {
                node->right = new_node;
                new_node->parent = node;
                return;
            } else {
                q.push(node->right);
            }
        }
    }
    
    /**
     * height() - 返回树的高度
     */
    int height() {
        return heightHelper(root);
    }
    
    /**
     * inorder() - 中序遍历
     */
    void inorder() {
        std::cout << "中序遍历: ";
        inorderHelper(root);
        std::cout << std::endl;
    }
    
    void inorderHelper(TreeNode* node) {
        if (node != nullptr) {
            inorderHelper(node->left);
            std::cout << node->key << " ";
            inorderHelper(node->right);
        }
    }
    
    /**
     * preorder() - 前序遍历
     */
    void preorder() {
        std::cout << "前序遍历: ";
        preorderHelper(root);
        std::cout << std::endl;
    }
    
    void preorderHelper(TreeNode* node) {
        if (node != nullptr) {
            std::cout << node->key << " ";
            preorderHelper(node->left);
            preorderHelper(node->right);
        }
    }
    
    /**
     * postorder() - 后序遍历
     */
    void postorder() {
        std::cout << "后序遍历: ";
        postorderHelper(root);
        std::cout << std::endl;
    }
    
    void postorderHelper(TreeNode* node) {
        if (node != nullptr) {
            postorderHelper(node->left);
            postorderHelper(node->right);
            std::cout << node->key << " ";
        }
    }
};

// 测试代码
int main() {
    BinaryTree bt;
    
    char keys[] = {'A', 'B', 'C', 'D', 'E', 'F', 'G'};
    for (char key : keys) {
        bt.insert(key);
    }
    
    std::cout << "树高度: " << bt.height() << std::endl;  // 2
    
    bt.preorder();   // A B D G E C F
    bt.inorder();    // D B G E A C F
    bt.postorder();  // G D E B F C A
    
    return 0;
}
```

### ❓ FAQ：树的遍历有哪些？分别什么时候用？

| 遍历方式 | 顺序 | 应用场景 |
|:--|:--|:--|
| **前序（Pre-order）** | 根 → 左 → 右 | 复制树、计算表达式、目录结构 |
| **中序（In-order）** | 左 → 根 → 右 | 二叉搜索树排序输出 |
| **后序（Post-order）** | 左 → 右 → 根 | 删除树、计算目录大小、依赖分析 |
| **层序（Level-order）** | 按层从上到下 | 最短路径、分层打印 |

### 🔑 关键概念

| 数据结构 | 操作特点 | 时间复杂度 | 典型应用 |
|:--|:--|:--|:--|
| **栈** | LIFO，后进先出 | push/pop: O(1) | 函数调用、括号匹配、撤销 |
| **队列** | FIFO，先进先出 | enqueue/dequeue: O(1) | 任务调度、BFS、缓冲 |
| **链表** | 灵活插入删除 | 查找: O(n)，增删: O(1)* | 频繁增删、LRU缓存 |
| **二叉树** | 层级结构 | 查找/插入: O(n)~O(log n) | 文件系统、表达式树 |
