## 💡 核心概念速览

### 算法是什么？
> 算法是任何良定义的计算过程，该过程取某个值或值的集合作为**输入**，并产生某个值或值的集合作为**输出**。

### 算法的关键特性
- **确定性**：每一步都有明确定义
- **有限性**：在有限步骤内完成
- **输入**：从外部获取
- **输出**：产生结果

### 时间复杂度等级
```
O(1)        < O(log n)  < O(n)       < O(n log n)
常数级别      对数级别     线性级别     线性对数级别

    < O(n²)       < O(n³)       < O(2ⁿ)
    平方级别       立方级别      指数级别
```
